# Python 코드 규칙 (AdaptiveAgent)

> 이 디렉터리(`adaptive_agent/`)와 그 하위 Python 파일에 적용되는 규칙이다.
> Claude Code는 이 디렉터리의 파일을 읽거나 편집할 때 이 `CLAUDE.md`를 자동으로 함께 로드한다.
> (Cursor의 `.cursor/rules/python.mdc` `globs: "**/*.py"`를 위치 기반으로 옮긴 것이다.)

- 핵심 로직과 공개 API의 docstring은 **문서 생성용 API 설명**으로 작성한다.
- docstring은 사용자에게 상황을 설명하는 말투가 아니라, 객체의 목적·계약·입출력·부작용을 간결하게 남긴다.
- 내부 구현을 보충하는 inline comment는 드물게 사용하고, 비자명한 정책·불변식·안전 제약만 설명한다.
- 테스트는 특정 구현 문구·스타일·한 함수 전용 출력에 과하게 맞추지 말고, 일반화 가능한 계약·동작·데이터 흐름을 검증한다.
- 프롬프트 `.txt`의 특정 문장, 주석 말투, identity 문구처럼 사용자가 자유롭게 바꿀 수 있는 표현은 테스트 고정값으로 두지 않는다.
- 한국어 입출력·파일 처리는 **UTF-8**을 사용한다.
